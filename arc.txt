 🧩 MODULE BREAKDOWN
--------------------------------------------------------------------------------

[ AI ENGINE ]
- Collaborative filtering & content-based recommendation logic.

[ BACKEND (API) ]
- FastAPI handles async requests for books, social feed, and auth.
- OTP services ensure secure user registration.
- Python-based scripts to create and seed the database.

[ FRONTEND (UI) ]
- Multi-language (AR/EN) React app with specialized portals for Admins/Students.
- Centralized component library for consistent UI across all pages.
- Built using Vite for module bundling and fast refresh.

[ DATA SERVICES ]
- Raw CSV files (Books details, ratings history).
